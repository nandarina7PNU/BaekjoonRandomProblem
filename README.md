# Baekjoon Random Problem

백준(BOJ) 문제를 티어 기준으로 랜덤 추천해주는 웹 프로젝트입니다.

## 프로젝트 설명

이 프로젝트는 solved.ac 데이터를 사용해 문제를 추천합니다.

- 티어 범위 뽑기: 선택한 티어의 5단계 전체에서 랜덤 1문제
- 세부 단계 뽑기: 선택한 티어의 특정 단계(1~5)에서 랜덤 1문제

예를 들어 gold를 선택하면 전체 골드 범위에서 뽑거나, Gold III 같은 특정 단계에서만 뽑을 수 있습니다.
